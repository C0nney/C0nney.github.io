f5之后可以看到代码还是很清晰的，这里简单提下涉及到的函数

- C 库函数 strcspn(str1, str2) 检索字符串 str1 开头连续有几个字符都不含字符串 str2 中的字符，返回值是字符数（数字）。

- C 库函数 strlen(str) 计算字符串 str 的长度，直到空结束字符，但不包括空结束字符，返回值是字符串长度。

- strncmp比较两个字符串的ASCII码，如str1=str2返回0，str1<str2返回值<0，str1>str2返回值>0。

- C 库函数 memcpy(*str1, *str2, n) 从存储区 str2 复制 n 个字节到存储区 str1，n为要被复制的字节数，返回一个指向目标存储区 str1 的指针。
